# Tutorial & Explainer Video Production

## When to Use
- How-to tutorials
- Educational content
- Product walkthroughs
- Process explanations
- Knowledge sharing

## Video Structure

### Tutorial Format (2-5 min)
1. **Hook & Promise (0-10s)** - What they'll learn
2. **Overview (10-30s)** - Brief roadmap of steps
3. **Step-by-step Content** - Main teaching body
4. **Summary (last 20s)** - Recap key points
5. **Next Steps/CTA** - What to do next

### Explainer Format (60-90s)
1. **Problem Statement (0-15s)** - Relatable pain point
2. **Introduce Solution (15-30s)** - Your approach
3. **How It Works (30-60s)** - Simple 3-step process
4. **Benefits (60-75s)** - Why this matters
5. **CTA (75-90s)** - Take action

## Visual Approaches

### Screen Recording Style
- Clean desktop with minimal distractions
- Cursor highlighting or zoom effects
- Clear annotations and callouts

### Animation Style
- Simple, clean motion graphics
- Consistent icon set
- Smooth transitions between concepts

### Presenter + Graphics
- Talking head with B-roll
- Lower thirds for key points
- Split screen for demonstrations

### Prompt Templates

**Educational Scene:**
```
Minimalist infographic illustration showing [concept],
clean vector style, soft colors, educational diagram,
simple icons, professional presentation style
```

**Process Visualization:**
```
Step-by-step workflow diagram, isometric illustration,
[step 1] to [step 2] to [step 3], modern flat design,
connecting arrows, numbered steps, clean background
```

## Best Practices
- One concept per section
- Visual for every major point
- Pause after key information
- Consistent visual language
- Clear audio/narration
