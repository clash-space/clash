# Social Media Short-Form Video Production

## When to Use
- TikTok content
- Instagram Reels
- YouTube Shorts
- Social media ads
- Viral content attempts

## Platform Specifications

| Platform | Aspect Ratio | Max Length | Sweet Spot |
|----------|--------------|------------|------------|
| TikTok | 9:16 | 10 min | 15-60s |
| Instagram Reels | 9:16 | 90s | 15-30s |
| YouTube Shorts | 9:16 | 60s | 30-45s |

## Viral Content Patterns

### Hook Types (First 1-3 seconds)
- **Question hook**: "Did you know...?"
- **Controversy**: "Unpopular opinion..."
- **Promise**: "This will change how you..."
- **Curiosity gap**: "I can't believe this actually..."
- **Visual shock**: Unexpected image/action

### Content Formats That Work
1. **Before/After** - Transformation content
2. **Day in the Life** - Relatable routines
3. **Tips/Hacks** - Quick value delivery
4. **Trending Audio** - Riding current trends
5. **POV** - First-person perspective stories
6. **Duets/Reactions** - Engaging with other content

### Engagement Triggers
- Ask questions in captions
- Use text overlays liberally
- Face in first frame increases retention
- Loop-worthy endings
- Pattern interrupts every 2-3 seconds

## Visual Style

### Native Feel (Recommended)
- Shot on phone aesthetic
- Natural lighting
- Authentic, unpolished vibe
- Quick cuts, jump cuts OK
- Text-heavy overlays

### Polished Feel (Brand content)
- Clean but not corporate
- Trending visual effects
- Platform-native transitions
- Captions burned in

### Prompt Templates

**Vertical Social Content:**
```
Vertical 9:16 composition, [subject] centered,
smartphone photography style, bright natural lighting,
social media aesthetic, engaging eye contact,
clean background, space for text overlay
```

**Trending Visual:**
```
[Trend description] style video frame,
vertical format, bold colors, high contrast,
Gen-Z aesthetic, meme-worthy composition,
attention-grabbing, scroll-stopping visual
```

## Platform-Specific Tips

### TikTok
- First 1s determines skip rate
- Use trending sounds (even at low volume)
- Green screen effects popular
- Stitch/Duet for reach

### Instagram Reels
- Slightly more polished than TikTok
- Aesthetic consistency matters
- Use Collab feature
- Share to Stories for extra reach

### YouTube Shorts
- Can be repurposed longer content
- Subscribe CTA at end
- Links in description work
- Thumbnail less important

## Common Mistakes
- Too corporate/polished
- Slow pacing
- No hook in first second
- Horizontal content in vertical
- Missing captions (80% watch muted)
