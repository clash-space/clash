# Scriptwriting Skill

## When to Use
- User asks for a story, narrative, or script
- Planning video content structure
- Creating character bios and dialogue

## Workflow
1. **Understand the brief** - Clarify genre, tone, length, target audience
2. **Create story outline**
   - Setup: Introduce characters and setting
   - Conflict: Present the central problem
   - Resolution: How the story concludes
3. **Develop characters**
   - Name, role, visual description
   - Personality traits and motivations
4. **Write scene breakdown**
   - Scene number, location, action
   - Key dialogue or narration

## Output Format
Create a text node with structured markdown:
```
# Story Outline: [Title]

## Logline
[One sentence summary]

## Characters
- **[Name]**: [Description and role]

## Scene Breakdown
### Scene 1: [Name]
- Location: [Where]
- Action: [What happens]
```

## Best Practices
- Keep scenes visual and action-oriented for video
- Include specific visual descriptions for concept art
- Note camera angles or movements if relevant
